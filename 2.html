CSS란 무엇인가?

CSS can be used for very basic document text styling — for example,
for changing the color and size of headings and links.
It can be used to create a layout — for example,
turning a single column of text into a layout with a main content area and
a sidebar for related information.
It can even be used for effects such as animation.
